Messaging App

I worked with a classmate to make a messaging application. This messaging application utilizes Amazon Web Services as a backend to put messages into a queue, send out notifications, and store message history. Our application uses the following AWS services: DynamoDB for the messaging history, Amazon SQS to send out messages, Amazon SNS to send notifications, and Amazon Lambda to save messages to history and queue notifications. This app also uses Electron to have a desktop app GUI made in HTML and CSS. It supports sending regular messages, links, images, and video. We named this application TwoLip, after mouths and the flower, and made a logo reflecting this. This project taught me a lot about cloud computing and AWS, as well as collaborating with others and persevering through challenges.

Dartmouth RC Calculator

During an internship at Dartmouth College I worked with their Research Computing team to make a webpage to allow their customers to calculate different compute jobs. Users could input different specifications for how much RAM, HDD space, and CPUs they want along with other preferences. The page would then give them the estimated cost per year and change the URL to include the inputs so that you could copy it and open the webpage with the same preferences and estimate. Completing this project required me to learn HTML, CSS, Javascript, and source code control with Github. During this internship I gained experience with communicating with other members of the team and collaborating remotely.

Quicksort

I wrote this program to run two ways. It could run through command prompt using command line arguments, or through a GUI. The GUI has an entry box that would search for a file of the inputted name in the same folder, or follow an inputted path. It also has a browse button that opens your file explorer and allows you to select the file and it automatically enters the path into the entry box. After pressing “Sort!”, it would then read and sort the file using the quicksort algorithm; it can read a single line with each number separated by a space, or one number per line. It would then overwrite the text in the same style it was inputted and return the time it took to sort.
